kyan XML Namespace

You found this page because our RSS feeds contain an URL that links here. Said URL is not an actual page but rather a unique identifier used to prevent name collisions with other XML namespaces.

The namespace contains the following additional, informational tags:

  • <kyan:seeders> holds the current amount of seeders on the respective torrent.

  • <kyan:leechers> holds the current amount of leechers on the respective torrent.

  • <kyan:downloads> counts the downloads the torrent got up to the point the feed was refreshed.

  • <kyan:infoHash> is the torrent's infohash, a unique identifier, in hexadecimal.

  • <kyan:categoryId> contains the ID of the category containing the upload in the form category_subcategory.

  • <kyan:category> contains the written name of the torrent's category in the form Category - Subcategory.

  • <kyan:size> indicates the torrent's download size to one decimal place, using a magnitude prefix according to ISO/IEC 80000-13.

  • <kyan:trusted> indicates whether the torrent came from a trusted uploader (YES or NO).

  • <kyan:remake> indicates whether the torrent was a remake (YES or NO).

  • <kyan:comments> holds the current amount of comments made on the respective torrent.
